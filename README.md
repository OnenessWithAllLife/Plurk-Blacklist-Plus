# Plurk 黑名單功能加強外掛（Chrome Extension）

目前噗浪的內建黑名單功能，只能讓你在自己的河道上看不到黑單帳號的主動發文。
當黑單帳號參與他人噗文時，你仍會看到該帳號的名稱與相關對話，影響精神衛生。

在此擴充功能中設定好黑名單帳號後，只要發言內容提及黑名單帳號（例如他人與黑單帳號對話並 ＠ 對方），就會自動隱藏該則討論與發言，類似噗浪的關鍵字隱藏功能，達到眼不見為淨的效果。

## Q&A

### Q：這個外掛與官方功能「不要在我的時間軸顯示以下類型的訊息」的差別？

A: 在開發這個專案以前已經做過多次測試，就算設定黑單帳號也無法隱藏他人回覆黑單帳號的留言，只好自己撰寫瀏覽器外掛。

### Q：可以只設定這個外掛的黑名單，不把想黑單的帳號放進官方黑名單功能嗎？

A：那還是會看到對方發言，對方也會看到你的發言。  
請先用官方黑名單功能設定黑單帳號，再用本外掛加強隱藏黑單帳號，捍衛精神衛生。

### Q：在隱藏留言前會閃一下 / 還是會有不到一秒的時間看到黑單帳號相關留言

A：此為已知問題。目前已盡可能以最快速度隱藏留言了，但還是沒法作到同官方功能等級完全看不到。

### Q：支援手機 App 嗎？

A：這個是 Chrome Extension，不支援手機版 App 或手機瀏覽器。我也很想要手機上有這功能。

## 安裝方式

有兩個方式可以選擇。

### 1. 以開發模式直接使用外掛

1. 開啟 Chrome 擴充功能頁面：
   ```
   open -a "Google Chrome" "chrome://extensions/"
   ```

2. 啟用「開發人員模式」（右上角開關）

3. 點擊「載入未封裝項目」

4. 選擇此專案資料夾

### 2. 打包為外掛檔再行安裝

```
mkdir -p dist
zip -r dist/plurk-blacklist-plus.zip . -x "dist/*" -x "*.git/*" -x "*.md"
```

## 使用說明

### 設定黑名單帳號
   - 點擊右上角外掛功能圖示 → 「開啟設定頁面」
   - 在輸入框中輸入帳號名稱（不需要輸入 `@` 符號）
   - 點擊「新增」，確認名單上有黑單帳號即可關閉設定頁面


### 進階設定說明

- **簡易隱藏模式**：啟用後會使用包含式比對，可能會有誤判風險（預設關閉）
- **過載時自動停用**：當處理時間過久時自動停用外掛，確保噗浪能正常運作（預設關閉）
