# 圖示檔案說明

此資料夾應包含以下圖示檔案：

- `16.png` - 16x16 像素（工具列圖示）
- `32.png` - 32x32 像素（Windows 高解析度）
- `48.png` - 48x48 像素（擴充功能管理頁面）
- `128.png` - 128x128 像素（Chrome Web Store）

## 建立圖示

您可以使用任何圖片編輯軟體建立圖示，或使用線上工具：

1. 設計一個簡單的圖示（建議使用禁止/隱藏相關的圖示）
2. 匯出為 PNG 格式
3. 調整尺寸並儲存到對應檔名

## 臨時解決方案

如果暫時沒有圖示，可以：

1. 使用任何 16x16 以上的圖片
2. 複製同一張圖片並重新命名為所有尺寸
3. Chrome 會自動縮放

注意：Chrome Web Store 上架時需要提供圖示檔案。
